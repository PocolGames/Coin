# 투자 수익률 계산기 기획서

## 구현 기능 목록

### 1. 입력 설정
- 시작날짜 입력 필드
- 시작값 입력 필드
- 증가율 입력 필드 (%)
- 목표값 입력 필드

### 2. 자동 계산
- 위 4개 값 입력 시 목표 달성까지 소요 일수 자동 계산
- 일별 예상 증가값 계산

### 3. 데이터 테이블
- 일차별 행 생성 (0일차부터 목표 달성일까지)
- 각 행 구성:
  - 일차
  - 예상값 (복리 계산)
  - 실제값 (사용자 입력 가능)
  - 수익률 (자동 계산)

### 4. 실제값 입력 기능
- 각 행의 실제값 컬럼 클릭하여 값 입력
- 엔터키 또는 포커스 아웃 시 저장

### 5. 수익률 자동 계산
- 실제값 입력 시 해당 일의 수익률 자동 계산
- 계산 공식: (실제값 - 전일값) / 전일값 × 100

### 6. UI 요소
- 모바일 반응형 테이블
- 입력 필드 검증
- 계산 결과 실시간 업데이트
